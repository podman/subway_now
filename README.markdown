# Subway Now
Subway Now is a simple command line client for ordering food from http://www.subwaynow.com

# Usage
## Required Gems

* mechanize
* text_highlight

## Commands
`./subway_now`
will display all of your favorites

`./subway_now -f 1`
will start your order with the given favorite number
